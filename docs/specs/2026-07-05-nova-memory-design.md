# Спека 3А: Память NOVA

Дата: 2026-07-05. Статус: одобрено в брейншторме, на ревью у Джея.
Этап 3 декомпозирован: 3А память (эта спека) → 3Б социнстинкт+звук
(заметки: 2026-07-05-social-instinct-notes.md) → 3В со-просмотр видео.

## Зачем

1. Амнезия реконнектов и смертных боксов: GPU крадут ~ежедневно, каждая
   новая сессия начинается с нуля («Ого, ты в Марвел Ривалз?» через
   минуту после часа совместного просмотра).
2. NOVA должна помнить жизнь: разговоры, события на экране (даже
   непрокомментированные), людей, игры, даты — и вспоминать детали
   недельной давности дословно, а не пересказом пересказа.

## Принципы (из брейншторма)

- **Дневник — истина, конспект — шпаргалка.** Сырой дневник никогда не
  сжимается и не удаляется; искажение краткостью допустимо только в
  конспекте, потому что за правдой всегда можно сходить в дневник.
- **Она летописец всей картины**: Джей — главный герой, но не
  единственный (друзья, игры, как играют, видосы, события).
- **Даты — гражданство памяти**: всё привязано к дням и времени.
- **Без цензуры**: конденсер — свой мозг (Qwen), не облако. Грязная
  шутка попадает в летопись дословно, в родном тоне.
- **Память бессмертна**: git-репо переживает любую кражу бокса.

## Хранилище: приватный репо `nova-memory`

```
diary/2026-07-05.md   # сырой дневник дня, append-only
digest.md             # скользящий конспект («рабочий стол»)
facts.md              # картотека фактов
index.md              # оглавление: одна строка-заголовок на день
```

Всё — человекочитаемый markdown; Джей может читать и править руками.

### diary/YYYY-MM-DD.md — сырой дневник

Строки с временем, тремя типами записей:

```
21:14 [Джей] Смотри, какой анлак!
21:14 [NOVA] Джефф только что съел троих — это не анлак, это дар.
21:16 [видела] Турнир Marvel Rivals: Джефф съел троих, счёт 0:1, овертайм.
```

- Реплики пишет оркестратор синхронно с диалогом (его/её, как услышано
  и как сказано — display-текст без ремарок).
- `[видела]` — визуальная летопись: описания кадров от Gemini-глаз
  пишутся В ДНЕВНИК и когда она промолчала (сейчас они выбрасываются).
  Так вспоминается и молчаливое («все промолчали про Джеффа»).
  ДВА источника: (а) по событиям детектора, как сейчас; (б) **летопись
  по пульсу** — каждые NOVA_CHRONICLE_S секунд (дефолт 45) при активном
  клиенте глаза фиксируют кадр в дневник, даже если событий нет: день —
  сплошная плёнка с шагом полминуты. GPU не участвует (только Gemini,
  ~$2-5/мес при плотной жизни); кэш кадров бережёт повторы.
- `[событие]` — системная канва: подключился/отключился, запустил
  игру/приложение, длительность сессии.

### digest.md — скользящий конспект

Градиент подробности: сегодня — детально, вчера — короче, неделя —
абзац на день, старше — строка на день (совпадает с index.md).
Бюджет: ~2500 токенов; конденсер обязан удерживать размер.
Каждый блок начинается с даты.

### facts.md — картотека

Разделы: **Люди** (Джей, Светка, ...), **Повадки и стили** — устойчивые
манеры поверх многих эпизодов: стиль игры («играет пацифистом, никого
не убивает»), реакции («в хоррорах орёт и ставит паузу»), вкусы в
фильмах/аниме/музыке, привычки в диалоге; **События**, **NOVA о себе**.
Повадки — топливо межконтекстных аналогий («ты и тут никого не
убиваешь, как тогда в X») — они всегда в голове (уровень 0), аналогия
рождается у мозга сама, без поиска по дневнику.
Факты НЕ сжимаются со временем: живут, пока актуальны; замена — новой
строкой с датой узнавания (`(с 05.07.2026)`), устаревшее конденсер
удаляет или помечает. Бюджет: ~1500 токенов.

### index.md — оглавление дневника

```
2026-07-05 ★: турнир Rivals, Джефф съел троих, спор про раскладку,
шутка про склероз | сущности: Rivals, Джефф, раскладка
```

Одна строка на день, пишет конденсер. Явные **сущности** (игры, люди,
NPC) — ключи для ассоциаций; **★** — эмоциональные пики дня (ржали,
орал, победа, эпичный фейл): ассоциации предпочитают яркое, как люди.
Топливо второго уровня recall и ассоциативного вспоминания.

## Запись

1. **Оркестратор → дневник**: каждая реплика и каждое описание кадров
   дописываются в diary сразу (локальный файл, мгновенно).
2. **Конденсер** (модуль memory/condenser): вызывается при тишине
   ≥10 мин И при остановке бокса (runner-хук). Берёт необработанный
   хвост дневника → одним вызовом Qwen: обновить digest (с учётом
   бюджета — ужать старое по градиенту), выделить новые/изменённые
   факты — ОСОБО охотиться за повадками и стилями (манера игры,
   реакции, вкусы: обобщения поверх эпизодов, не только события) —
   и обновить строку index за день. Отметка «докуда дневник уже
   обработан» — служебный файл .cursor в репо памяти.
   - Реплика пользователя ПРЕРЫВАЕТ конденсер (asyncio cancel),
     доделает в следующую паузу. Джей всегда главнее конспекта.
   - Запасной рубильник NOVA_MEMORY_LLM=gemini (без изменения кода
     промптов), дефолт — мозг.

## Чтение: recall() — трёхуровневое вспоминание

Уровень 0 — всегда: digest.md + facts.md в системный промпт каждой
сессии (лечит амнезию реконнекта само по себе).

Уровень 1 — триггер: слова «помнишь», «тогда», «вспомни», «на прошлой
неделе», «недавно», «в прошлый раз», «когда-то» в реплике →
`recall(вопрос)`:
  а) парс дат из фразы («1-2 недели назад» → диапазон дней);
  б) выбор дней-кандидатов: по диапазону + по совпадению слов вопроса
     со строками index.md (простая релевантность, difflib/пересечение
     слов — без эмбеддингов в v1);
  в) из diary выбранных дней — куски вокруг совпадений ключевых слов
     (окно ±5 строк), суммарно ≤1500 токенов;
  г) вставка мозгу: `[из дневника за 21.06: ...]` — и обычный ответ.

Уровень 1.5 — **ассоциативное вспоминание («эффект Светки»)**: без
всякого вопроса, при КАЖДОЙ генерации (ответ или коммент) анализатор
пробегает index с ключами текущего момента — название игры, имена с
экрана (из [видела]-описаний), темы последних реплик. Сильное
совпадение со старым днём (fuzzy-матч по сущностям, порог; ★-дни в
приоритете; чем старше при сильном матче — тем интереснее) → кусок
дневника подкладывается мозгу с пометкой «если к месту — можешь
вспомнить». Мозг решает сам: ввернуть «а помнишь, 4 месяца назад этот
NPC...» или промолчать.
  - **Защита от бабушкиных историй**: журнал вспомненного
    (memory/.recalled) — одна и та же байка не всплывает чаще, чем раз
    в N дней. Уметь НЕ вспоминать — тоже человечность.
  - Стоимость: локальный поиск по строкам index — миллисекунды, ноль
    лишних вызовов моделей (кандидаты подкладываются в уже идущий
    вызов мозга).
  - Морфология: «Джефф/Джеффа/Джеффом» — нечёткое сравнение слов
    (difflib, порог ~0.75), тот же приём, что в срезе шапки голоса.

Уровень 2 (будущее, вне этой спеки): эмбеддинг-поиск по смыслу, когда
index перестанет влезать. Интерфейс recall() не меняется.

## Синхронизация и выживание

- Бокс: `git clone` nova-memory в onstart (до старта сервера);
  push — **после каждого обмена репликами** (фоновой задачей, диалог
  не ждёт; схлопывается, если обмены чаще раза в ~30с) + при остановке
  (runner-хук). Окно потери при краже бокса — одна последняя фраза.
  Дневник append-only + один пишущий бокс — конфликты практически
  исключены; при любом конфликте побеждает объединение (rebase).
- Ноут: `git pull` в NOVA_START.bat — локальная копия всегда свежая.
- Токен: fine-grained PAT с правом contents:write ТОЛЬКО на
  nova-memory (не на код NOVA!), лежит в .env (NOVA_MEMORY_TOKEN) и
  на боксе в /workspace/memory_token. Худший случай кражи бокса:
  чужой читает/пишет летопись — приемлемо по решению Джея.
- Потеря при краже: максимум последняя фраза (см. пуш выше).

## Что заполняется, а что нет (рационально)

- Диск/git: ~10-20 МБ в год — не заполнится никогда практически.
- Контекст мозга: держится бюджетами digest (2.5К) + facts (1.5К) +
  recall-вставка (1.5К) — суммарно ~5.5К из 32К, остальное диалогу.
- Старое не «забывается» — уходит с рабочего стола в дневник, откуда
  достаётся recall'ом дословно.

## Критерии приёмки

1. Реконнект/пересоздание бокса: она продолжает разговор со знанием
   дня и не переспрашивает то, что уже было.
2. «Помнишь, недели две назад Джефф съел троих?» — вспоминает деталь,
   которой нет в конспекте (из дневника, включая [видела]-записи).
2а. Эффект Светки: при возвращении в игру/тему из прошлого она МОЖЕТ
   сама ввернуть старый момент («этот NPC тогда ляпнул...») — и не
   повторяет одну байку каждый вечер.
2б. Межконтекстная аналогия по повадке: замеченный месяцы назад стиль
   («никого не убиваешь») всплывает в ДРУГОЙ игре/теме к месту —
   через факт-повадку, без поиска по дневнику.
2в. Гармония: ассоциации и аналогии — предложения мозгу, не приказы;
   она не молчит вечерами и не сыплет воспоминаниями без остановки
   (частота регулируется, финальная калибровка — разметкой Джея в 3Б).
3. Грязная шутка попадает в дневник и конспект дословно.
4. Конденсер не задерживает ответы: реплика во время сжатия отвечается
   без заметной паузы (сжатие прервано).
5. После недели жизни digest ≤ бюджета, index растёт по строке в день,
   git-репо содержит всё.
6. `uv run pytest -q` зелёный; юниты: store (append/бюджет), recall
   (даты из фраз, выбор дней, куски), condenser (формат промпта,
   прерывание), orchestrator-интеграция на моках.
